# Python练习题集合

## 文件说明

本文件夹包含了为你准备的Python练习题，按照难度和知识点分为8个练习：

### 基础巩固练习（适合当前水平）

1. **练习1_猜数字游戏改进版.py**
   - 知识点：random模块、循环、条件判断、break/continue
   - 难度：⭐⭐
   - 预计时间：30分钟

2. **练习2_学生成绩管理系统.py**
   - 知识点：函数、列表、循环、条件判断
   - 难度：⭐⭐⭐
   - 预计时间：45分钟

3. **练习3_打印各种图形.py**
   - 知识点：嵌套循环、print控制、字符串操作
   - 难度：⭐⭐⭐
   - 预计时间：40分钟

4. **练习4_密码验证系统.py**
   - 知识点：while循环、input、条件判断、break
   - 难度：⭐⭐
   - 预计时间：20分钟

5. **练习5_计算器程序增强版.py**
   - 知识点：函数、while循环、异常处理
   - 难度：⭐⭐⭐
   - 预计时间：35分钟

### 进阶练习（面向对象）

6. **练习6_银行账户管理系统.py**
   - 知识点：类、对象、方法、属性
   - 难度：⭐⭐⭐⭐
   - 预计时间：50分钟

7. **练习7_图书管理系统.py**
   - 知识点：类、字典、文件操作
   - 难度：⭐⭐⭐⭐
   - 预计时间：60分钟

### 综合项目练习

8. **练习8_学生信息管理系统.py**
   - 知识点：面向对象、文件操作、异常处理、综合应用
   - 难度：⭐⭐⭐⭐⭐
   - 预计时间：90分钟

## 学习建议

### 学习顺序
1. 先完成基础巩固练习（练习1-5）
2. 再完成进阶练习（练习6-7）
3. 最后挑战综合项目（练习8）

### 每个练习的完成步骤
1. 仔细阅读练习要求
2. 查看代码提示
3. 自己编写代码
4. 测试运行
5. 如有问题，可以参考小甲鱼课程和林粒粒课程的相关知识点

### 学习时间安排
- **基础巩固练习**：建议每天完成1-2个练习，一周内完成
- **进阶练习**：建议每天完成1个练习，一周内完成
- **综合项目练习**：建议分2-3天完成
